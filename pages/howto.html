Apresentação do Html

O html é uma Linguagem de Marcação HiperTexto muito usado na área de informática pela sua praticidade e facilidade. E também muitos programadores recomendam o html para os iniciantes para terem uma lógica de Programação.

No html usa se tags que ficam dentro dos sinais (<) (>), vamos falar deles no próximo parágrafo. Um exemplo deles são h1, html, body, e etc.

Tags

A tag html é a primeira tag que se utiliza no seu projeto para iniciar a programar. A tag body vem antes da head, e também a tag body vai representar o corpo da página.

a tag h1 até a h6 são tags de títulos de texto para organização do texto da página.

Css

O Css é nada mais nada menos que o responsável por dar uma vida para página, você pode colocar fontes de letras maiores de outros tipos, adicionar cor a página entre outros.
